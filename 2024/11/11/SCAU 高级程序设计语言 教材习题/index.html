18041 分期还款(加强版)

Description

从银行贷款金额为 d，准备每月还款额为 p，月利率为 r 。请编写程序输入这三个数值，计算并输出多少个月能够还清贷款，输出时保留 1 位小数。 如果无法还清，请输出 “God” 计算公式如下：

$$ m = \frac{\log{\frac{p}{p - d \times r}}}{\log{(1+r)}} $$

输入格式

三个数，分别为货款金额、每月还款和月利率，以空格分隔，均为非负数，其中 d，p，r >  = 0

输出格式

需要还款的月份

输入样例

输出样例

题解

主要是使用 if-else筛去无法计算或者不符合实际的情况，其余的情况正常计算即可

  1. 1 + r > 0 时，无法计算，输出 God
  2. p − d × r = 0 时，该分数无意义，无法计算，输出 God
  3. p 与 p − d × r 符号不一致时，$\frac{p}{p - d \times r} \le 0$ 无法计算，输出 God （由于题目给出 p 是非负的 ，只需要判断 p 是否等于 0 和 p − d × r 是否小于 0 即可）
  4. 当 d = 0 时，无需还款，直接输出 0.0
  5. 其他情况，直接计算即可
